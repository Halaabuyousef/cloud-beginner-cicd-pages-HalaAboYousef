What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment. It is a modern software development practice that automates the process of building, testing, and deploying code. CI ensures that code changes are integrated and verified frequently, while CD automates the delivery of these changes to the production environment, such as GitHub Pages, ensuring a fast and reliable release cycle.

My Pipeline

Once I push my code to the main branch on GitHub, a GitHub Actions workflow is triggered automatically. It verifies that all essential files exist and then deploys the latest version of the website to GitHub Pages without any manual intervention.
